Dominio web

Dominio web (domain en inglés) es una dirección o nombre alfanumérico único que se escoge para un sitio o página web, que nos sirve para buscarlo e identificarlo. Este también nos puede dar pauta del sector o categoría, a donde va enfocado el producto o servicio.

Toda página de internet es única por su domino web. Gracias a la gran cantidad de portales web que existen es importante tener un nombre que sea solamente tuyo. Para que de esa forma sea fácil identificar.

El dominio web, te proporciona eso. Ya que este término se usa para establecer el nombre que posee tu página web. Este nombre es exclusivo y ninguna otra persona podrá tomarlo, dándote así seguridad.

¿Cuál es la estructura y funcionalidad del dominio web?

El dominio web posee una estructura muy sencilla de entender. Consta de tres partes: la primera es el World Wide Web (www), la segunda el nombre de la página. Por ultimo, el tipo de la organización a la que pertenece, esto sería .com, .org, etc.

El dominio no sólo da un nombre particular, sino que traduce tu dirección IP.

Esta abstracción hace posible que cualquier servicio (de red) pueda moverse de un lugar geográfico a otro en la red Internet, aun cuando el cambio implique que tendrá una dirección IP diferente. — wikipedia.org

Sin esto se tendría que colocar el IP, el cual contiene muchos números y letras para memorizar. De igual manera provocaría que existieran menos páginas ya que algunas contienen las mismas direcciones IP.

¿Cuáles son los tipos de dominio web?

Existen diversos tipos de domino web que te ayudan no sólo a reconocerte sino a saber qué tipo de organización eres. Los tres principales son: Genérico, Territorial y de tercer Nivel.

Genérico:

Este tipo de dominio se emplea para un uso común y no representa algún país. Sólo determina el concepto de tu página web. Entre los destacados están: .com (representa sitios comerciales), .net (empresas de servicio de internet), .org (organización sin fines de lucros).

Territorial:

Es conocido también como dominio territorial. Es usado para identificar el país y se componen siempre por dos letras. Algunos ejemplos de este tipo de dominio web son: .mx (México), .co (Colombia), .es (España), .uk (Reino Unido).

Tercer nivel

Este domino es la combinación de los dos anteriores, es decir del genérico y el territorial. Este tipo de dominio delimita la página a su mismo territorio. Algunos ejemplos serían: .com.mx (sitio comercial en México) .com.es (sitio comercial en España)

También existen los subdominios, estos permiten la modificación del www. Éste se coloca del lado izquierdo y sirve tanto para clasificar, administrar o dar un acceso directo a una dirección de internet. Un ejemplo de este caso sería: https://es. (Página en español), https://fr. (Página en francés)

¿Cuál es su historia?

Años atrás, no existía este dominio, sino que los portales web se identificaban con un archivo llamado hosts.txt. Esto permitía que todas las computadoras estuvieran interconectadas a la red y de esa forma ser conocidas. Pero, al tiempo se dieron cuenta que ese método no funcionaba.

Por eso, los ingenieros Paul Mockapetris y Jon Postel decidieron idear un sistema de dominios.

Este sistema permitiría asociar los nombres de dominio previamente registrados, a direcciones IP de los ordenadores conectados de una forma más sencilla. El dominio se utiliza para “humanizar” la Red y hacer la navegación más sencilla y accesible para las personas. — Rubén Andrés, computerhoy.com

Gracias a esto no sólo se pudo identificar de forma más sencilla los portales de internet. Sino que se pudo catalogar y estructurar de mejor manera las páginas web.

¿Para qué sirve el dominio web?

Logrará que por medio del nombre que elijas, que las personas puedan localizarte y memorizar tu dirección. Es más fácil aprenderse tu dominio que tu dirección de IP. Todo esto para que sea más simple reconocerte y de esa manera obtener más visitas en tu contenido de internet.
